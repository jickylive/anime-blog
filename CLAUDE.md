## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run server
# or
npx hexo server

# Generate static files
npm run build
# or
npx hexo generate

# Clean and regenerate
npm run clean
# or
npx hexo clean
```

## Architecture Overview

This is a **Hexo-based static blog** deployed as a subdirectory (`/blog/`) of the main hxfund.cn site.

- **Framework**: Hexo 8.1.1
- **Theme**: defaultone (custom theme based on hexo-theme-redefine)
- **Features**: Live2D character animations, particles.js background, PWA support
- **Deployment**: GitHub Actions в†’ Aliyun virtual host (`/htdocs/public/blog/`)

## Key Commands

| Command | Description |
|---------|-------------|
| `npm run server` | Start Hexo dev server |
| `npm run build` | Generate static files (`npx hexo generate`) |
| `npm run clean` | Clean generated files |
| `npm run deploy` | Deploy to remote |
| `npm run news` | Generate daily news content |
| `npm run news:proxy` | Generate news via proxy |
| `npm run check` | Health check script |
| `npm run dns` | DNS setup script |

## Creating New Posts

```bash
# Create a new post
npx hexo new post "Post Title"

# Create a new page
npx hexo new page "About"

# Create a draft
npx hexo new draft "Draft Title"
```

Post front-matter template:
```yaml
---
title: Post Title
date: 2024-03-21 12:00:00
tags: [tag1, tag2]
categories: [Category]
---
```

## Configuration

### Site Config (`_config.yml`)

Key settings:
- `url: https://www.hxfund.cn/blog`
- `root: /blog/`
- `theme: defaultone`
- `language: zh-CN`

### Theme Config (`themes/defaultone/_config.yml`)

Controls theme appearance, menu, colors, and features like:
- Live2D character settings
- Particles.js background
- Comment system (Waline)
- SEO settings

## Deployment

### GitHub Actions
Push to `main` branch triggers automatic deployment via FTP to Aliyun virtual host.

Required secrets:
- `FTP_HOST`, `FTP_USER`, `FTP_PASS`, `FTP_PORT`

### Manual Deploy
```bash
npx hexo clean
npx hexo generate
# Upload public/ directory to /htdocs/public/blog/
```

## Important Notes

1. **Private deployment only**: Not for commercial use (check theme license)
2. **Node.js >= 16.0.0** required
3. **Output directory**: `public/` is gitignored, generated by `hexo generate`
4. **Deploy path**: Files go to `/htdocs/public/blog/` (subdirectory of main site)
